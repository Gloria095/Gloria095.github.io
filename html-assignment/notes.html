<body> </body>

The body element contains all the content of an HTML document, such as headings, paragraphs, images, links, and more.

<head></head>

The head element contains meta information about the document, such as the title of the page, which is displayed in the browser's tab, and link to other resources like CSS files.

<div> </div>

Is a block-level element used for grouping HTML elements and applying styles to those elements. It provides a way to structure the content of a web page into sections and does not have any semantic meaning, meaning it does not describe the content it contains.

<h1> </h1>

Is a heading tag used to create the main heading of a web page. It is typically used as the title or top-level heading of a document, and it is usually the largest text on the page. The tag has semantic meaning, meaning it describes the content it contains and helps to create a clear outline for the structure of the web page.

<p> </p>

The paragraph element is used to create paragraphs of text on a web page.
